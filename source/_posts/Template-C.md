###Template语法
```
< templates >
```
templates可为如下几种类型：
1. class T/typename T
```
<class T> 
 or 
<typename U>
```  

2. A data type 

3. An integral specification

4. An integral constant/pointer/reference which maps to spication mentioned above.

###Template的应用类型主要是以下两种:

- Function Template 

- Class Template
